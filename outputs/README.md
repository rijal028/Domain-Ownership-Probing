# Outputs

This folder stores generated results from running DomainProbe.

Typical outputs:
- `results.json`
- `heatmap.png`
- `ownership_bar.png`
- `cohesion_bar.png`

Note:
- This folder may be empty in the GitHub repository.
- Files inside `outputs/` are generated locally after running `python run.py`.
- Optional: you can ignore large output files in `.gitignore` if needed.
